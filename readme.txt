Automatically generate beautiful HTML and PDF newsletters from your weekly blog posts.

== Description ==

Weekly Post Newsletter is a powerful WordPress plugin that automatically collects your weekly blog posts and transforms them into professionally designed newsletters in both HTML and PDF formats.

== Features ==

* Automatic weekly post collection
* Beautiful HTML newsletter generation
* PDF newsletter creation
* Customizable templates
* User-friendly admin interface
* Preview functionality

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/weekly-post-newsletter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Weekly Newsletter screen to configure the plugin

== Frequently Asked Questions ==

= How often are newsletters generated? =

The plugin collects posts from the last 7 days when you generate a newsletter.

= Can I customize the newsletter template? =

Yes, you can customize both HTML and PDF templates by editing the template files in the plugin's templates directory.
